# College Attendance Management System

A web-based Attendance Management System developed as a BCA Final Year College Project using PHP and MySQL.  
The system provides Admin and Student panels to manage attendance, subjects, teachers, and reports with a clean dark UI.

---

## Project Modules

### Admin Panel
- Admin Login and Forgot Password (OTP Verification)
- Admin Dashboard with Statistics
- Manage Admins, Students, Teachers and Subjects
- Edit and Delete Records
- Take and Manage Attendance
- Export Attendance Reports to Excel
- Feedback Management

### Student Panel
- Student Login and Dashboard
- View Subjects and Attendance
- Attendance Reports
- Profile and Feedback System

---

## Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Library: PhpSpreadsheet (Excel Export)
